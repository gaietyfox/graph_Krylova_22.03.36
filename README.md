# Радиальный график токенов по корпусу IELTS Writing Task 1

## Описание

В работе представлен радиальный график, в котором:

- слова располагаются по кругу
- размер слова зависит от его частоты
- разные типы графиков выделены разными цветами
- из центра к словам идут линии-лучи

Согласно заданию, график намеренно сделан "нечитабельным" для пользователя :)

## Файлы проекта

- `graph_ielts_Krylova_22.03.26.R` — основной R-скрипт
- `corpus_parsed_clean.csv` — корпус для анализа
- `Krylova_22.03.26.png` — итоговая визуализация

## Данные

Для построения графика используется файл `corpus_parsed_clean.csv`, который взят из моего проекта 1 

## Используемые библиотеки

В проекте используются следующие пакеты R:

- `ggplot2`
- `dplyr`
- `readr`
- `tidyr`
- `stringr`

## Как запустить

1. Скачайте файлы проекта
2. Откройте `graph_ielts_Krylova_22.03.26.R` в RStudio
3. Убедитесь, что файл `corpus_parsed_clean.csv` находится в той же папке
4. Установите нужные пакеты, если они ещё не установлены
5. Запустите скрипт

## Результат

На выходе создаётся радиальный график токенов, в котором:

- отображаются 3 наиболее частотных типа `chart_type`
- для каждого типа выбираются наиболее частотные токены (топ-15)
- визуализация сохраняется в PNG-файл
